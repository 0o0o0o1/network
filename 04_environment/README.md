# 환경 변수

## 1. 환경 변수란?

환경 변수(Environment Variable)는 컴퓨터에서 실행 중인 프로세스의 동작 방식에 영향을 미치는 값이다.  
사용자가 직접 정의할 수 있으며, 시스템과 프로그램이 필요한 정보를 전달하는 데 사용된다.

환경 변수는 일반적으로 **키-값(Key-Value) 쌍** 형태로 구성된다.

---

## 2. 사용 목적

환경 변수는 다음과 같은 정보를 저장하는 데 사용된다.

- 실행 파일 경로
- 사용자 정보
- 시스템 설정
- 프로그램 설정 (예: API Key)

---

## 3. 주요 환경 변수

| 변수 | 설명 |
|------|------|
| PATH | 실행 파일을 찾는 디렉터리 경로 |
| HOME | 사용자 홈 디렉터리 |
| LANG | 시스템 언어 설정 |
| PWD | 현재 작업 디렉터리 |
| USER | 현재 사용자 이름 |
| SHELL | 기본 쉘 |
| MAIL | 메일 저장 위치 |
| UID | 사용자 ID |
| LS_COLORS | ls 출력 색상 설정 |

---

## 4. 핵심 정리

- 환경 변수는 프로세스 동작에 영향을 준다.
- 키-값 형태로 구성된다.
- 시스템 및 프로그램 설정에 사용된다.
