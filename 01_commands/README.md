# Basic Commands

## 1. 개념
리눅스는 CLI(Command Line Interface)를 기반으로 동작하며  
명령어를 통해 파일과 디렉토리를 직접 제어할 수 있다.

처음에는 명령어가 많아서 헷갈렸지만, 자주 사용하는 것 위주로 반복하니 점점 익숙해졌다.

---

## 2. 주요 명령어 정리

| 명령어 | 설명 |
|--------|------|
| ls | 디렉토리 목록 확인 |
| cd | 디렉토리 이동 |
| pwd | 현재 위치 확인 |
| cp | 파일 복사 |
| mv | 파일 이동 |
| rm | 파일 삭제 |
| mkdir | 디렉토리 생성 |
| touch | 파일 생성 |
| cat | 파일 내용 출력 |
| clear | 화면 정리 |
