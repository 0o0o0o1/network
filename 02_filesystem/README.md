# File System

## 1. 개념
파일 시스템은 데이터를 저장하고 관리하기 위한 구조이며  
리눅스에서는 디렉토리가 트리 구조로 구성된다.

실제로 사용해보니 현재 위치를 정확히 아는 것이 중요했다.

---

## 2. 파일과 디렉토리

- 파일 : 데이터를 저장하는 기본 단위  
- 디렉토리 : 파일을 담는 공간  

리눅스에서는 대부분의 요소가 파일 형태로 관리된다.

---

## 3. 경로(Path)

- 절대경로 : /home/user/file.txt  
- 상대경로 : docs/file.txt  

처음에는 상대경로가 헷갈렸는데, 현재 위치를 기준으로 생각하면 이해하기 쉬웠다.

---

## 4. 주요 디렉토리

- / : 최상위 디렉토리  
- /home : 사용자 작업 공간  
- /etc : 설정 파일  
- /var : 로그 및 캐시  
- /usr : 프로그램  

---
